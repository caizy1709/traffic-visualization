# 出租车轨迹可视化系统

系统中使用的数据是经过了轨迹压缩算法和聚类算法的，展示的是处理之后的数据。

## 安装和启动

项目使用 [thinkjs 2.2](https://thinkjs.org/zh-cn/doc/2.2/index.html) 版本开发

```sh
git clone https://github.com/fuyi501/traffic-visualization.git
cd traffic-visualization-web
npm install
npm start
```
